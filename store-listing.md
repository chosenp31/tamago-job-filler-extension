# Chrome Web Store 掲載情報

## 拡張機能名
Tamago-DB 求人自動入力

## 概要（132文字以内）
求人サイトの求人情報をワンクリックでコピーし、Tamago-DBの新規求人登録フォームに自動入力する拡張機能です。

## 説明
求人サイトに掲載されている求人情報を、Tamago-DBの新規求人登録フォームに自動で転記する拡張機能です。

【使い方】
1. 求人情報が表示されているページで、拡張機能のアイコンをクリック
2. 「求人情報をコピー」ボタンを押す
3. Tamago-DBの新規求人フォームを開く
4. 拡張機能のアイコンをクリックし「Tamago-DBに貼り付け」ボタンを押す

【自動入力される項目】
- タイトル（職種/募集ポジション）
- ヘッドライン（求人タイトル）
- 雇用タイプ
- 給与（下限・上限・詳細）
- 勤務地（都道府県・市町村）
- 応募資格
- 福利厚生
- 仕事内容

【特徴】
- ワンクリックでコピー＆ペースト
- コピーしたデータはタブを切り替えても保持されます
- 入力された項目は緑色にハイライトされ、どこが入力されたか一目で確認できます
